## Fitur
- Tambah data mahasiswa  
- Lihat daftar mahasiswa  
- Update data mahasiswa  
- Hapus data mahasiswa  
- Terhubung ke database MySQL  
- Tampilan sederhana berbasis JSP

## Teknologi yang Digunakan

- Java EE (JSP/Servlet)  
- MySQL Database  
- Apache Tomcat  
- NetBeans IDE  
- mysql-connector
